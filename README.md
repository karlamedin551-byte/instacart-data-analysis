# Análisis de Datos de Instacart

## Descripción del Proyecto
Análisis exploratorio de datos de la plataforma de entregas de comestibles Instacart. Este proyecto examina los patrones de compra de los clientes, productos más populares, y comportamientos de reorden.

## Estructura del Proyecto
```
Proyecto-Instacart/
├── instacart_analysis.ipynb
├── datasets/
│   ├── instacart_orders.csv
│   ├── products.csv
│   ├── aisles.csv
│   ├── departments.csv
│   └── order_products.csv
└── README.md 
```
## Principales Hallazgos
- Las horas pico de pedidos son entre las 10:00 AM y 4:00 PM
- Los domingos y lunes son los días más populares para hacer compras
- Las bananas son el producto más popular y reordenado
- El 50% de los productos que compran los clientes son reordenados
- Los productos orgánicos dominan las ventas
